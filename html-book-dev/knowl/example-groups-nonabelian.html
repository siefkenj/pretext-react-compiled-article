Example 3.2.6. Symmetries of a Triangle is not Abelian.

The symmetries of an equilateral triangle described in Section 3.1 form a nonabelian group. As we observed, it is not necessarily true that \(\alpha \beta = \beta \alpha\) for two symmetries \(\alpha\) and \(\beta\text{.}\) Using Figure 3.1.7, which is a Cayley table for this group, we can easily check that the symmetries of an equilateral triangle are indeed a group. We will denote this group by either \(S_3\) or \(D_3\text{,}\) for reasons that will be explained later.
