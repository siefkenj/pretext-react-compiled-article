Exploration 4.2. Exploring Explorations.
This is an <exploration>. Other similar possibilities are <project>, <activity>, <task>, and <investigation>.
Note that projects, activities, explorations, tasks and investigations share the independent numbering scheme, so it is really only intended you use one of these. If you want a variant of the name (e.g. “Directed Activity”) you can use the <rename> facility (Subsection 30.1).
