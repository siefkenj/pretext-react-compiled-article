Because of the Fundamental Theorem‚Äâ1‚Äâ, for every derivative we know, there is an antiderivative we might find useful. Because of the Fundamental Theorem of Calculus‚Äâ2‚Äâ, we recycle the ‚Äú\(\int\)‚Äù symbol as notation for an antiderivative.
  • Derivatives
    1. \(\displaystyle \frac{d}{dx}x^n = nx^{n-1}\)
    2. \(\displaystyle \frac{d}{dx}e^x = e^x\)
    3. \(\displaystyle \frac{d}{dx}\cos(x) = -\sin(x)\)
  • Antiderivatives
    1. \(\displaystyle \indefiniteintegral{x^n}{x} = \displaystyle\frac{x^{n-1}}{n+1}\text{ if }n\neq -1\)
    2. \(\displaystyle \indefiniteintegral{e^x}{x} = e^x\)
    3. \(\displaystyle \indefiniteintegral{\sin(x)}{x} = -\cos(x)\)
